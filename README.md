Создать Платформу LMS со следующим функционалом:
- Создание курсов администратором системы, назначения преподавателя на курс. 
- Преподаватель формирует группы из пользователей, отмечает посещаемость, открывает и закрывает темы. 
- Учащиеся имеют доступ только к своему курсу. Могут изучать материал курса, присылать домашнее задание, в виде файла или ссылки на облачный диск
